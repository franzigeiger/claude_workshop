Findings

01AGENTS.md / CLAUDE.md👍 Solid
AGENTS.md names the stack and lists five quotable non-obvious rules: idempotent mutations, no swallowed errors, pathlib.Path everywhere, stdout-only logging, env-only ANTHROPIC_API_KEY. CLAUDE.md is a symlink. Missing: a ## Commands section — agent has to grep mise tasks to find mise run check.
02Strict type / compiler settings🚀 Ship-Ready
pyproject.toml [tool.mypy]: strict = true plus warn_unreachable, warn_redundant_casts, warn_unused_ignores, disallow_any_explicit. mypy src tests runs in mise check and on pre-commit.
03Strict linter / formatter🚀 Ship-Ready
Ruff selects E, W, F, I, B, UP, SIM, RUF, TID, PL, PTH, ARG, ERA — well beyond defaults. ruff format --check and ruff check both gate mise run check. Sensible per-file ignores for tests.
04Schema validation at boundaries➖ N/A
Pydantic is a declared dependency but no API/CLI surface exists yet. design.md §11 specifies a JSON envelope; once the CLI lands that envelope is the thing to model with Pydantic.
05Logic separated from I/O➖ N/A
src/paperclaw/__main__.py is a 13-line scaffold (logging setup + "scaffold ready" message). There is no business logic in the tree yet to evaluate.
06One-command bring-up🚀 Ship-Ready
.mise.toml defines setup (uv sync + .env copy + lefthook install), check (format · lint · mypy · deptry · pytest), and fmt. Same verbs work from anywhere. README mirrors them.
07Pre-commit feedback loop👍 Solid
lefthook.yml covers gitleaks, ruff format, ruff check --fix, mypy on pre-commit and pytest on pre-push. Caveat: .git/hooks/ currently contains only .sample files — hooks won't fire until mise run setup is run. Standard but worth a one-line nudge in AGENTS.md.
08Dead-code guardrail🚀 Ship-Ready
Ruff F401 (unused imports) and F841 (unused locals) on; deptry src in mise check catches orphan dependencies. Currently passes against the scaffold.
09Logs reachable from terminal🚀 Ship-Ready
logging.basicConfig(stream=sys.stdout, ...) at src/paperclaw/__main__.py:7. AGENTS.md explicitly forbids FileHandler. An agent running the CLI sees everything.
10Docs stay in sync with code🩹 Patchy
content/context.md instructs the agent to keep project.md and design.md current, but nothing enforces it — no lefthook check, no CI rule. design.md is already aspirational (Chroma + Ollama + tesseract pipeline) while the code is a 13-line main. Drift on day zero.
Fix: add a lefthook pre-commit rule that warns when src/**/*.py is staged without a matching content/**/*.md touch.
11Agent can self-test end-to-end🩹 Patchy
uv run paperclaw exists, is documented in README, and writes to stdout — wiring is fine. But the binary only logs "paperclaw scaffold ready"; there is no subcommand the agent can drive to verify a change. Today it's a stub.
Fix: stub paperclaw doctor first (per design.md §11) so the agent has a real entrypoint that returns something meaningful while the pipeline is being built.
12Agentic review panel💀 Broken
No REVIEW.md, no /review slash command, no mise run review, no panel instructions in AGENTS.md. Every diff routes through a single human reviewer.
Fix: add REVIEW.md with an ignore-list, plus a mise run review task that fans out 3 specialist Claude reviewers (security, python-idiomatic, maintainability) over git diff main...HEAD, tiered by line count.
13Friction proportional to blast radius💀 Broken
No danger-zone hook, no CODEOWNERS, no named bypass, no marked surfaces in AGENTS.md. The agent could silently change the ~/library/ mutation path (the irreversible-on-disk surface this whole tool exists to manage) with no extra speed bump.
Fix: mark src/paperclaw/store/** as a danger zone in lefthook.yml pre-push, requiring an explicit PAPERCLAW_LIBRARY_OK=1 bypass, and print a checklist of what to verify first.
14Tooling tuned for the agent👍 Solid
.gitleaksignore is present and ready (empty, but the channel exists for false-positives). Mise task names are readable. Gap: lefthook.yml commands exit non-zero on failure without printing the exact remediation command — e.g. "run mise run fmt" or "run uv run mypy src tests locally".

🎁 Bonus finds

  • disallow_any_explicit in mypy — beyond --strict; closes the Any escape hatch agents reach for under deadline.
  • mise run setup bundles lefthook install — bring-up genuinely one command; no "now also run lefthook" footgun for the next agent that clones.
  • Concrete, quotable AGENTS.md rules — "check destination before writing", pathlib.Path not strings, env-only API key. Specific enough to act on, not generic platitudes.
